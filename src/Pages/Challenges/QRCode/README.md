### What I learned

I discovered two methods of centering an element vertically

```html
<div class="flex w-full h-screen">
  <div class="flex justify-center m-auto">
    <!--  content goes here-->
  </div>
</div>
```

```html
<div class="flex w-full h-screen items-center justify-center">
  <!--  content goes here-->
</div>
```
